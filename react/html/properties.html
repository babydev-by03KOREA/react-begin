여기서 잠깐! STYLE == {{이중 중괄호}}?

아닙니다! Style을 바로 선언하고 컴포턴트 안에서 사용하면 style={{fontSize: '30px'}}이고, 변수값으로 가져오려면 {중괄호 한번}입니다!
