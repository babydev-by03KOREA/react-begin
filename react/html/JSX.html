JSX

BABEL은 JAVASCRIPT 문법을 확장해주는 도구입니다. 과거 IE지원에 많이 사용되었습니다. 아직 지원하지 않는 최신 문법을 정식 JS로 변환해주는 작업을 합니다.

즉, JSX 코드는 Babel을 이용하여 React.createElement(...)로 변환하여 브라우저에서 실행이 가능하다는 것 입니다!

다음 규칙을 반드시 지켜주세요!

  • 태그를 열었으면 꼭 닫아주어야 합니다.
  • 두개 이상의 태그는 무조건 하나의 태그로 감싸져야 합니다.
